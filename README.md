# Web3 Chess (Firebase)

Minimal multiplayer chess app using React + Firebase (Anonymous Auth + Firestore).

## Run

```powershell
cd client
npm install
npm run dev
```

## Setup

1. Copy `client/.env.example` to `client/.env`
2. Fill Firebase web config values
3. Enable Anonymous Auth and Firestore in Firebase Console
4. Publish `client/firestore.rules`

More details: see `QUICKSTART.md`.
